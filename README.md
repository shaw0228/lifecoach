# Life Coach AI 助手

一个温暖治愈的 AI 生活教练，帮助你解决生活中的困扰。

## 功能特点

- 💬 智能对话：基于 DeepSeek R1 模型的智能对话系统
- 🎵 治愈音乐：内置轻音乐播放功能
- 🎨 温暖界面：粉色系温柔设计
- 💾 本地存储：自动保存对话历史
- 🌐 实时响应：流式对话体验

## 技术栈

- Frontend: React + Material-UI
- Backend: Express + SQLite
- API: DeepSeek R1
